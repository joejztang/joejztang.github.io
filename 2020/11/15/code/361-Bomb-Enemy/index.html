2020-11-15
Contents

  2. 2. Analysis

Description

Given a 2D grid, each cell is either a wall ‘W’, an enemy ‘E’ or empty ‘0’ (the number zero), return the maximum enemies you can kill using one bomb.
The bomb kills all the enemies in the same row and column from the planted point until it hits the wall since the wall is too strong to be destroyed.
Note: You can only put the bomb at an empty cell.

Example:

Analysis

I can think of a naive solution immediately, that is checking every position and calculate how many enemies that I can eliminate.
Time would be O(mn(m+n)), and Space would be O(1)

This is also a DP problem.

There are four directions, left to right, right to left, top to bottom, bottom to top.
Standing at a certain position, given a certain direction, I will know how many enemies I can kill. Adding 4 direction together and find the largest.

Time O(mn)
Space O(mn)
