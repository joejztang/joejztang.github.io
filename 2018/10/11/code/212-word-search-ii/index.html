212. word search II

2018-10-11

description

Given a 2D board and a list of words from the dictionary, find all words in the board.

Each word must be constructed from letters of sequentially adjacent cell, where “adjacent” cells are those horizontally or vertically neighboring. The same letter cell may not be used more than once in a word.

Example:

Note:
You may assume that all inputs are consist of lowercase letters a-z.

analysis

  1. find character in board that being same with the first character in words array.
  2. dfs to see if it can be found, if found then add to res.

comments:

  1. use hashtable to record word.charAt(0) -> word. if word is found, then delete this mapping in hashtable (de-duplicate purpose).
  2. once found, break the for loop and LET the function backtracking. if just simply return, then will not find any other words.
