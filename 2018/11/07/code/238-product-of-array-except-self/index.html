238. product of array except self

2018-11-07

description

Given an array nums of n integers where n > 1, return an array output such that output[i] is equal to the product of all the elements of nums except nums[i].

Example:

Note: Please solve it without division and in O(n).

Follow up:
Could you solve it with constant space complexity? (The output array does not count as extra space for the purpose of space complexity analysis.)

analysis

Your first instinct is to multiply everything and use division. You can’t do that though. But I came up with this answer pretty quickly without seeing it before though it might have been luck. They want constant space so you know you can’t store anything so one way or another you can only store the product of the numbers. I was looking at it visually to see if I could see a pattern and I thought “what if I ‘dropped off’ the product of every number I see along the way?”:

Then I just asked myself “what are they missing to be correct?”

When laid out this way the answer was obvious.
